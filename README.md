# Task Manager ✨

Task Manager is a lightweight, modern task management web application built with HTML, CSS, and JavaScript. It provides an intuitive interface for creating, organizing, and managing your daily tasks. With a sleek glassmorphism design and smooth animations, Task Manager enhances your productivity while keeping the experience visually appealing.

## Features

- **Add Tasks:** Insert new tasks at the top or bottom of the list.
- **Task Reordering:** Move tasks up or down within the list.
- **Mark as Complete:** Toggle task completion status, which disables reordering and moves completed tasks to the bottom.
- **Delete Tasks:** Remove tasks with a single click.
- **Responsive Design:** Optimized for both desktop and mobile devices.

## Installation

Follow these steps to get a local copy of the project up and running:

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/your-username/task-manager.git
   ```

2. **Navigate to the Project Directory:**

   ```bash
   cd task-manager
   ```

3. **Open the Application:**

   Open the `index.html` file in your preferred web browser.

> **Note:** No additional dependencies are required. The project leverages a Google Font (Inter) for styling, which is loaded via the HTML.

## Usage

1. **Launch the App:**
   - Open `index.html` in your browser.

2. **Add a Task:**
   - Type a task description into the input field.
   - Click **"Add to Top"** to insert the task at the beginning or **"Add to Bottom"** to add it at the end.

3. **Manage Tasks:**
   - **Reorder:** Use the up (↑) and down (↓) buttons to change the task order.
   - **Complete:** Click the check (✓) button to mark a task as completed. Completed tasks are visually distinct and cannot be reordered.
   - **Delete:** Remove tasks by clicking the delete (×) button.

## Folder Structure
